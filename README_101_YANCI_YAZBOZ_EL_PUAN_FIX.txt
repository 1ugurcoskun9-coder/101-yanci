101-yanci v1.4.9 Yazboz El Puan Fix

Yapılan değişiklik:
- Deste bittiğinde yeni ele geçmeden önce mevcut elin puanları yazboza işlenir.
- Yazboz popup içinde 1. El / 2. El satırlarında puanlar görünür.
- Toplam satırı şimdilik boş bırakıldı; toplam hesaplama kurallar tamamlanınca bağlanacak.
- Kimse el açmadıysa oyunculara 202 yazılır.
- Oyuncu seri açtıysa elde kalan taşların toplamı yazılır.
- Oyuncu çift açtıysa elde kalan taşların toplamının 2 katı yazılır.
- Yeni test başladığında yazboz el puanları sıfırlanır.

Not:
Bu patch sadece el bitişi -> yazboz puan yazma -> yeni ele geçiş akışını düzenler.
